# Project Name
> Interns 


## General Information
- This project demostrate how to Create, Read, Update, and Delete from the local database 

## Technologies Used
- Room persistence library
- Coroutines and Flow
- Jetpack Navigation
- Jetpack ViewModel
- Jetpack LiveData



## Features
- Create, Read, Update, and Delte items in the database
